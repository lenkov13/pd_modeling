### Проектная идея
- Реализация автоматизированного решения, позволяющего на основе рыночных данных получать PD-кривую для оценки кредитных рисков эмитентов

### Цель
- на основе анализа рыночных данных по облигациям, акциям и деривативам (опционы на акции эмитента) оценить применимость моделей структурного подхода, редуцировнного, моделей с неполной информацией и пр.

### Результат 
- доклад / статья по методологии построения PD кривой, проблемы реализации тех или иных моделей на российском рынке
- Базовый SDK для калибровки моделей к рыночным данным, построения PD кривых, анализа кредитного риска отдельного эмитента
