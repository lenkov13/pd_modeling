# Задача 1

_Попробовать провести репликацию подходов ЦЦ НРД в части построения z-spread эмитентов_

В методике НРД (файл Metodika_NRD) представлен подход к построению кривой z-spread эмитентов (Раздел 4 Методики), в который входит:
1. Построение Модели Нельсона-Сигеля-Свенсона (NSS)
2. Использование Фильтра Калмана, который позволяет
  - Оценивать изменяющиеся во времени параметры кривой z-спредов
  - Учитывать историческую динамику параметров
  - Сглаживать рыночный шум в наблюдениях
  - Работать с неполными данными (пропущенными наблюдениями)

Для начала предлагаем следующее:
1. Выбрать несколько эмитентов из разных рейтинговых групп и выгрузить данные по всем имеющимся облигациям у данных эмитентов. Желательно удалить из выборки облигации с офертой (put/call), бессрочные облигации и облигации с плавающим купоном (пока что). То есть работать с plain vanilla bonds.
2. Облигации должны быть активными. Стандартные критерии активности - 1) не менее 10 сделок за 1 каледарный месяц? 2) не менее 5 дней, в течение которых были сделки + объем торгов / объем номинала в обращении >= 0.1%)
3. Рыночную информацию можно найти на официальном сайте [ПАО Московская Биржа](https://www.moex.com/S2644), а также на сторонних сервисах, например [СмартЛаб](https://smart-lab.ru/q/bonds/). При необходимости можем помочь с выгрузкой.
4. Для теста можно построить файлик в Excel, для дальнейшей работы желательно автоматизировать с помощью написания кода.
